# 01 — 项目愿景

## 核心目标

打造一款**像素风格、双摇杆操作、高重玩价值**的 Roguelike 地牢冒险手游，目标 **2026 年 Q3 上架 App Store / Google Play**，定价 **$2.99 买断 + 可选 DLC**。

## 一句话定位

> "在地牢中成长、死亡、重生 —— 每次冒险都不一样。"

## 立项背景

| 项目 | 内容 |
|------|------|
| 开发者 | 1 人（独立开发） |
| 启动资金 | $99（开发者欠款本金） |
| 引擎 | Flutter + Flame（轻量、跨平台、独立友好） |
| 周期 | 2026/05 - 2026/09（5 个月） |
| 平台 | iOS 主，Android 同步发布 |

## 设计哲学

1. **核心循环要爽** — 30 秒能上手，3 分钟体会乐趣，30 小时还在玩
2. **死亡是一切** — 死亡推动成长，每次失败都让下一次更强
3. **随机但不混乱** — 房间、武器、天赋随机，但难度曲线必须可控
4. **手机优先** — 横屏双摇杆，单手撑过 5 分钟也能玩
5. **像素美学** — 像素不是省钱借口，是风格选择

## 目标体验

玩家打开游戏后的第一次成功体验：

1. 选英雄（10 秒）
2. 进入第一层（5 秒过场）
3. 干掉一个敌人（5 秒，第一次开火很爽）
4. 清空第一个房间（30 秒）
5. **三选一升级**（决策瞬间，建立"我在变强"的认知）
6. 进入第二个房间，更难一点
7. 吃到武器升级（"哇这把枪能连射"）
8. 死在第三层（"再来一次！"）

每一步都要给玩家明确的反馈和成长感。

## 成功指标（KPI）

| 指标 | 目标 |
|------|------|
| App Store 评分 | ≥ 4.3 |
| 用户次日留存 | ≥ 40% |
| 用户 7 日留存 | ≥ 20% |
| 平均单次游戏时长 | ≥ 8 分钟 |
| 平均每用户游戏次数 | ≥ 15 次 |
| 首月下载量 | ≥ 5000 |
| 付费转化率 | ≥ 3% |
| 还清 $99 债务的时间 | ≤ 2 个月 |

## 不做什么（避免范围蔓延）

- ❌ 多人联机
- ❌ 3D 画面
- ❌ 开放世界
- ❌ 复杂剧情和过场动画
- ❌ 内购解锁付费道具（避免 P2W 差评）
- ❌ 强制广告（破坏沉浸感）
- ❌ 账号系统、云存档（首版本）
