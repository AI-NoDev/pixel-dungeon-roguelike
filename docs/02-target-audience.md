# 02 — 目标受众与市场定位

## 核心受众画像

### 主要用户：硬核手游玩家（70%）

- **年龄**：18-35 岁
- **性别**：男性偏多（约 65%）
- **设备**：iPhone / 中高端 Android
- **付费习惯**：愿意为单机精品付 $3-10
- **游戏偏好**：玩过元气骑士、合金弹头、以撒、枪火重生
- **游玩时长**：单次 5-15 分钟，每天 2-4 次（碎片化）

### 次要用户：休闲玩家寻求挑战（25%）

- **年龄**：25-45 岁
- **特征**：上下班通勤、午休时间消遣
- **决策点**：评分 ≥ 4.5、截图好看、价格 ≤ $4

### 长尾用户：Roguelike 爱好者（5%）

- **特征**：会写 Reddit 长测评、上 YouTube 录视频
- **价值**：UGC 传播、口碑发酵
- **运营**：Discord 社群、关卡设计建议征集

## 区域市场策略

### 主战场：中国大陆

- 最大手游市场，对像素 Roguelike 接受度高
- TapTap、4399、好游快爆等渠道资源
- 风险：版号问题（首发选择 App Store 海外，不上国内安卓）

### 次战场：北美/欧洲

- 用户付费意愿强
- 评分体系成熟，好游戏容易出圈
- TouchArcade、Pocket Gamer 等媒体覆盖

### 第三梯队：日本/韩国/东南亚

- 后续考虑增加 JP/KR 本地化
- 通过 Apple "本日推荐" 等编辑推荐渗透

## 用户获取渠道

### 免费渠道（重点）

- Reddit（r/roguelikes、r/iosgaming）
- TapTap、好游快爆
- YouTube/Bilibili 评测视频投放（1 个 KOL）
- App Store 编辑推荐（联系 Apple 媒体团队）
- TouchArcade、IndieDB 投稿

### 付费渠道（备选）

- 上架后第一周不投放，先看自然流量
- 评分稳定后才考虑 ASA（Apple Search Ads）
- 单次 CPI 控制在 ≤ $0.5

## 核心卖点（写在 App Store 描述）

1. **真正的 Roguelike** — 每次都不一样
2. **元素反应系统** — 火+冰=蒸发，玩出花样
3. **4 个英雄 4 种玩法** — 骑士、游侠、法师、盗贼
4. **2.99 美元买断** — 无内购、无广告、无氪金
5. **中英双语** — 完整本地化
6. **离线可玩** — 不需要网络
